# Agent History
Projeto criado para gerenciar e documentar historicos de agentes de LLM. O projeto foi realizado em React + Vite, com adições do daisyUI e TailwindCSS e plugins adicionais como o React Code Editor. 

O projeto se encontra na BETA 1, há uma funcionalidade mínima, mas outras funções e melhorias de interface serão introduzidas nas próximas betas. 

**Funcionalidades:**
- Criação de fluxos
- Criação de agentes
- Criação de históricos
- Criação de diferentes tipos de prompt em um mesmo histórico
- Importação/Exportação dos dados
- Dados salvos localmente no navegador
- Busca e filtro de fluxos e agentes
- Editor de código para o prompt funcional e com suporte a linguagens populares para prompts
- Ícone personalizado para os fluxos
